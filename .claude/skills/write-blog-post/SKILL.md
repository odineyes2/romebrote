# 블로그 글 작성

주어진 주제로 블로그 글 초안을 작성한다.

## 규칙

- 분량: 800~1200자 내외
- 톤: 친근하고 실용적인 말투, 너무 광고처럼 들리지 않게
- 구조: 후킹 인트로 → 본문(소제목 3~4개) → 마무리 요약

## 작업 순서

1. blog-writer 서브에이전트에게 주제를 전달해 초안을 작성하게 한다.
2. 초안이 drafts/에 저장되면, blog-reviewer 서브에이전트에게 검토를 요청한다.
3. 승인되면 종료. 문제가 있으면 피드백을 blog-writer에게 전달해 한 번 더 수정시키고, 다시 검토받는다.
4. 최대 2회 수정 후에도 통과 못하면 남은 문제를 사람에게 보고하고 멈춘다.
